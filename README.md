Project Name
CI/CD Course Assignment
Description
This project demonstrates the implementation of a CI/CD pipeline using modern DevOps tools and practices. It covers the key concepts of Continuous Integration and Continuous Delivery to automate the software development lifecycle.
Tools Used

GitHub Actions
Tekton
OpenShift Pipelines
Docker

Pipeline Stages

Plan - Define project requirements and tasks
Code - Write and review application code
Build - Compile and package the application
Test - Run automated tests to ensure quality
Deploy - Release the application to production

Key Concepts

Infrastructure as Code (IaC)
Social Coding
GitOps
Continuous Monitoring
